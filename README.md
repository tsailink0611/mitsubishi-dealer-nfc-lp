# 三菱自動車ディーラー 営業マン向けNFC名刺LP

NFCプラスチックカード名刺をスマホでタッチすると開くランディングページ。
主目的はLINE友達追加による継続的な顧客接点の確立。

---

## デザインテーマ

**温かみ系**（基本パターン）
- カラー：三菱レッド × ディープネイビー × クリーム白
- フォント：Noto Serif JP（見出し）+ Noto Sans JP（本文）

---

## 差し替えポイント一覧

`index.html` を開いて、コメント `<!-- 差し替えポイント -->` の箇所を変更してください。

| 項目 | 変更内容 |
|---|---|
| LINE ID | `@YOUR_LINE_ID` を実際のIDに（3箇所） |
| 担当者名 | `佐藤 健太` / `Kenta Sato` |
| 店舗名 | `三菱自動車 仙台北店` |
| 役職 | `カーライフコンサルタント` |
| プロフィール写真 | コメントを解除して `YOUR_PHOTO_URL` に画像URLを設定 |
| キャリア年数 | `<strong>8</strong>` の数値 |
| 自己紹介文 | `about-message` クラスのテキスト |
| 得意車種バッジ | `about-badge` クラスの要素 |
| キャンペーン内容 | 各 `campaign-card` のテキスト |
| ギャラリー写真 | コメントを解除して `YOUR_GALLERY_*_URL` に設定 |
| YouTube動画ID | `YOUR_YOUTUBE_VIDEO_ID` を実際の動画IDに（コメント解除） |
| 住所 | `宮城県仙台市泉区〇〇 1-2-3` |
| 電話番号 | `022-000-0000`（2箇所） |
| 営業時間 | 平日・土日祝・定休日 |
| Googleマップ | コメントを解除して `YOUR_GOOGLE_MAPS_EMBED_URL` に設定 |
